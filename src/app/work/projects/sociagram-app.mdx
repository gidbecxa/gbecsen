## Overview

Sociagram is an all-in-one platform designed to help **content creators, influencers, business owners**, and everyday users boost their social media engagement. The app provides essential tools for **discovering trending content, crafting compelling captions, and optimizing post visibility with hashtags**.

The project was developed using **React Native (Expo)** for the mobile app and **Supabase (PostgreSQL)** for the backend, handling **user authentication, data storage, and content management** seamlessly. With a strong focus on **performance, usability, and real-time content updates**, Sociagram empowers users to stay ahead of social media trends.

This project highlights my expertise in **React Native (Expo) and Next.js**, along with **database management in Supabase (PostgreSQL), authentication, and cloud storage handling**.

## Challenges and Learnings

### **Optimizing Data Retrieval for a High-Performance User Experience**
- Initially, fetching large datasets for **hashtags and captions** led to slow responses.
- Optimized using **Supabase Edge Functions and indexed queries**, improving speed by **50%**.

### **Handling Real-Time Content Updates Efficiently**
- **Database triggers and subscriptions** were implemented to update UI without polling.
- Reduced API calls, improving both **performance and battery efficiency**.

### **Ensuring Secure Content Storage & User Privacy**
- Implemented **Supabase Storage policies** to restrict unauthorized file access.
- **Row-Level Security (RLS)** ensures users can only access their own saved content.
