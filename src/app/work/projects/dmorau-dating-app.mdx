## Overview

**DMORAU** is a next-generation dating platform designed to enhance user engagement through **smart matchmaking, in-app gifting, business integrations, and real-time messaging**. The system was built with a focus on scalability, security, and a seamless user experience.

The project showcases my expertise in **React Native (Expo), Nest.js, and Next.js**, with deep integration of AWS services for backend deployment, PostgreSQL for robust data management, and Stripe for handling transactions.

## Technologies Used

### **Frontend (React Native - Expo)**

- **Expo 51**: Ensures a smooth, cross-platform experience.
- **NativeWind**: Provides a clean and scalable styling system.
- **WebSockets**: Powers real-time messaging.

### **Backend (Express.js)**

- **Express.js**: Provides a modular and scalable backend.
- **PostgreSQL on AWS RDS**: Optimized relational database.
- **AWS EC2**: Live backend deployment.
- **Stripe API**: Secure payment processing.
- **Twilio API**: Real-time OTP for phone number verification.
- **WebSockets**: Powers real-time messaging.

## Challenges and Learnings

### **Optimizing the Matchmaking Algorithm**

- Built an **efficient query system** for fetching matches in real-time.
- **Indexed database fields** to improve search performance.

### **Ensuring Secure Transactions**

- Implemented **Stripe webhooks** for payment verification.
- Secure **JWT authentication** and **session management**.

### **Scalability in Real-Time Messaging**

- **WebSockets** optimized for thousands of concurrent users.
- **Message queues** ensure efficient data processing.
